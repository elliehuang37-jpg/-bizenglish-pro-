## 🚀 上傳步驟（完全免費）

### 第一步：註冊 GitHub 帳號
1. 開啟 https://github.com
2. 點右上角 Sign up
3. 輸入 Email、密碼、使用者名稱

### 第二步：建立新專案
1. 登入後點右上角 ＋ → New repository
2. Repository name 填：`bizenglish-pro`
3. 選 **Public**（必須選Public才能免費使用Pages）
4. 勾選 **Add a README file**
5. 點 **Create repository**

### 第三步：上傳檔案
1. 在新專案頁面，點 **Add file → Upload files**
2. 把以下檔案全部拖進去：
   - index.html
   - manifest.json
   - sw.js
   - icons/ 資料夾（整個拖入）
3. 在下方 Commit changes 輸入：`Add BizEnglish Pro PWA`
4. 點 **Commit changes**

### 第四步：開啟 GitHub Pages
1. 點上方 **Settings** 標籤
2. 左側選單找 **Pages**
3. Source 選 **Deploy from a branch**
4. Branch 選 **main**，資料夾選 **/ (root)**
5. 點 **Save**
6. 等約 1-2 分鐘，頁面會出現：
   > ✅ Your site is live at: https://你的名字.github.io/bizenglish-pro/

## 📲 手機安裝步驟

### iPhone / iPad（Safari）：
1. 手機 Safari 開啟上方的網址
2. 點底部 **分享按鈕（□↑）**
3. 向下滑找到 **「加入主畫面」**
4. 點 **新增** → 桌面出現 💼 APP 圖示！

### Android（Chrome）：
1. 手機 Chrome 開啟網址
2. 右上角點 **三個點選單**
3. 選 **「新增到主畫面」** 或 **「安裝應用程式」**
4. 點安裝 → 桌面出現 APP 圖示！
